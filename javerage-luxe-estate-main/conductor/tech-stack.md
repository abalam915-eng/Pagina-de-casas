# Technology Stack: LUXU Real Estate

## 1. Core Languages
- **TypeScript:** Typed JavaScript for enhanced maintainability and error-checking.

## 2. Frontend Framework & Libraries
- **Next.js 16.1.6:** Utilizing the App Router for robust server-side rendering, routing, and optimizations.
- **React 19.2.3:** Core library for building user interfaces.

## 3. Styling & UI Components
- **Tailwind CSS 4+:** Utility-first CSS framework for rapid styling.
- **Shadcn/UI:** Accessible, customizable UI component library.

## 4. Specific Feature Libraries
- **Leaflet & React-Leaflet:** For interactive map rendering of properties.

## 5. Backend & Data Infrastructure
- **Supabase:**
  - **Database:** PostgreSQL for robust relational data storage.
  - **Authentication:** OAuth integrations (Google, GitHub) for the administrative portal.
  - **Storage:** Supabase Storage Buckets for handling property images.
